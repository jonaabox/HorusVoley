Diseño — Feature #4: Modales de confirmación

Hook useConfirm + componente ConfirmModal

Archivos nuevos

src/hooks/useConfirm.js
Hook que maneja estado interno del modal. Devuelve { confirm, ConfirmModal }.

Uso en cualquier página:
const { confirm, ConfirmModal } = useConfirm()

Llamada (reemplaza el confirm() nativo):
const ok = await confirm({ title: '...', message: '...' })
if (!ok) return
src/components/ConfirmModal.jsx
Acepta props: title, message, confirmLabel, variant (danger / info), onConfirm, onCancel.

Variant danger → botón rojo "Eliminar" (para los 3 confirm)
Variant info → botón primario "Entendido" (para el alert de teléfono)

Pagos.jsx → Reemplaza confirm() en handleDelete
Campanas.jsx → Reemplaza confirm() al eliminar grupo
